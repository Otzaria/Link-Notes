# מקשר הערות אוטומטי

כלי סטטי (קובץ HTML בודד, ללא שרת ותלויות) שמייצר קבצי `_links.json` לקישור בין קובץ טקסט (ספר) לקובץ הערות, עבור תוכנת [אוצריא](https://www.otzaria.org/).

[כניסה לאתר](https://otzaria.github.io/Link-Notes/)

## איך זה עובד

מסמנים בכל אחד משני הקבצים (ספר והערות) את מספרי ההפניה — לרוב בתגית כמו `<sup>1</sup>`, בדיוק במקום שבו אמורה להופיע ההפניה. הכלי מזהה את המספרים אוטומטית, מתאים כל מופע בספר לשורה המתאימה בהערות (פרק אחר פרק), ומייצא קובץ JSON בפורמט שאוצריא מצפה לו.

## תכונות עיקריות

- **זיהוי לפי תגית או ביטוי רגולרי (regex)** — ברירת המחדל היא תגית HTML (כמו `sup`), אך אפשר להזין גם ביטוי רגולרי בשדה התג (למשל `\([א-ת]{1,3}\)` עבור סימון כמו "(א)"), והכלי יזהה זאת אוטומטית.
- **שלושה מצבי יצירת קישור** — רגיל (שורה↔שורה), עוגן-מילה אוטומטי (קישור מילה אחת או יותר בתוך השורה, עם `start`/`end`), וטווח אוטומטי (קישור כמה שורות רצופות, עם `line_index_1_end`/`line_index_2_end`). ניתן גם לשלב עוגן וטווח על אותו קישור.
- **עורך מתקדם** — תצוגה חיה של שלושה טורים (ספר / הערות / JSON) לצפייה, עריכה ותיקון ידניים של קישורים, עם הדגשה זמנית (flash) של קישורים קיימים בלחיצה, וכפתור איפוס לחזרה לתוצאה האוטומטית.
- **לוגים מפורטים לדיבוג** — מסוף המפתחים (F12) מציג בדיוק אילו הערות לא קושרו אוטומטית ומדוע (מזהה לא נמצא כלל, עיטור שונה, יישור פרקים שגוי וכו'), לצד סיכום מלא של כל ריצה.
- **עבודה אופליין** — ניתן להוריד עותק עצמאי של הדף לשימוש ללא חיבור לרשת.
- **תמיכה מלאה במובייל ובמסכים צרים.**

## שימוש מהיר

1. ייבוא קובץ הספר וקובץ ההערות.
2. הגדרת תג (או ביטוי רגולרי) לזיהוי בכל צד, וסוג הקישור.
3. לחיצה על "יצירת ה-JSON".
4. בדיקת התוצאה, ובמידת הצורך תיקון ב"עורך מתקדם".
5. הורדת קובץ ה-JSON וטעינתו באוצריא > הגדרות > ייבוא דורות וקישורים.
